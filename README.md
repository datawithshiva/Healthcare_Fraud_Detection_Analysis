🏥 Healthcare Fraud Detection Analysis

📌 Project Overview

Healthcare fraud leads to significant financial losses and inefficiencies in the insurance system. This project analyzes healthcare claims data to identify fraudulent activities using Python, SQL, and Power BI.

The objective is to detect fraud patterns by evaluating claim behavior, provider activity, patient demographics, and financial discrepancies, and provide actionable insights to improve claim validation and reduce losses.

🎯 Business Problem

Healthcare organizations struggle to identify:

Fraudulent insurance claims

High-risk providers and hospitals

Overbilling and claim manipulation

Fraud trends across time and demographics

👉 This project solves these challenges using a data-driven approach.

📊 Dataset Summary

Rows: ~4,000+

Columns: 14+

Key Data Includes:

Patient details (Age, Gender)

Provider & hospital information

Claim details (Claim Amount, Approved Amount)

Insurance type

Fraud flag (0 = Genuine, 1 = Fraud)

⚙️ Tools & Technologies

🐍 Python (Pandas, NumPy, Seaborn, Matplotlib)

🗄️ SQL (PostgreSQL)

📊 Power BI

📁 GitHub

🔍 Exploratory Data Analysis

Cleaned and preprocessed data (null values, duplicates, formatting)

Created new features like Claim Difference and time-based features

Analyzed:

Claim amount distribution

Provider and hospital behavior

Demographic trends

🧠 SQL Analysis (Business Questions)

Total fraud cases and fraud percentage

Top fraudulent providers and hospitals

Average fraud claim amount

Insurance types with highest fraud

Fraud distribution by age group

High claim difference fraud detection

Monthly fraud trends

Length of stay comparison (fraud vs genuine)

📊 Power BI Dashboard

Built an interactive dashboard to:

Track Fraud KPIs (Fraud %, Total Claims, Avg Claim Amount)

Identify high-risk providers & hospitals

Analyze fraud trends over time

Detect high claim discrepancies

Enable filtering by insurance type, provider, and time

💡 Key Insights

High claim differences indicate potential overbilling fraud

A small group of providers contribute to major fraud cases

Certain insurance types are more fraud-prone

Fraud patterns vary across age groups and months
